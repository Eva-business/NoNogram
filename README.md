# 🧩 Pixel Nonogram Puzzle Game

A pixel-style Nonogram puzzle game built with **Flutter**.

Players solve picture puzzles by filling grid cells according to the numeric hints provided for each row and column.

This project demonstrates a complete puzzle game implementation including puzzle generation, custom puzzle upload, persistent records, and animated UI.


## 🎮 Features

### Classic Nonogram Gameplay
Solve puzzles by filling cells according to row and column number hints to reveal hidden pixel images.

### Built-in Puzzle Library
Includes several built-in puzzles with different difficulty levels.

### Custom Puzzle Mode
Players can upload their own images from the gallery and generate Nonogram puzzles automatically.

### Record Gallery
Completed puzzles are stored and can be viewed in the record gallery.

### Pixel Style UI
Retro pixel-style user interface designed for a puzzle game experience.

### Background Music
The game includes background music that can be toggled on and off.

### Animated Home Screen
The home screen uses **Lottie animations** to create an interactive and dynamic UI.

# 🧩 像素數織益智遊戲

這是一款使用 **Flutter 開發的像素風數織益智遊戲**。

玩家需要根據每一行與每一列的提示數字填滿格子，逐步完成隱藏的像素圖案。

本專案展示了一個完整的數織遊戲實作，包括題庫系統、圖片轉關卡、玩家自訂關卡、紀錄收藏以及動畫化的使用者介面。


## 🎮 遊戲特色

### 經典數織玩法
依照行列提示數字填滿格子，逐步完成隱藏的像素圖案。

### 內建關卡題庫
內建多個像素圖關卡，並依難度分類。

### 玩家自訂關卡
玩家可以從相簿上傳圖片，系統會自動將圖片轉換成數織題目。

### 紀錄館
完成的關卡會被收藏到紀錄館中，可以隨時查看。

### 像素風格 UI
採用復古像素風格的介面設計，營造遊戲氛圍。

### 背景音樂
遊戲內含背景音樂，並可在首頁開啟或關閉。

### 動畫首頁
首頁使用 **Lottie 動畫**，增加互動與視覺效果。
